# Wildberries Antibot Solver (`x_wbaas_token`)

Скрипт для автоматического прохождения антибот-защиты сайта Wildberries. Он извлекает необходимые ключи, взаимодействует с локальным сервисом для решения челленджей (solver) и генерирует валидный cookie-файл `x_wbaas_token` для выполнения успешных запросов к маркетплейсу.

---

## Требования

* Установленный Python версии 3.7 или выше.
* Библиотека `requests` для выполнения HTTP-запросов.
* Наличие файла `fingerprint.json` с валидным слепком браузера в корневой директории проекта.

---

## Установка и запуск

1. Скачайте файл `wb.py` в рабочую директорию вашего проекта.
2. Установите необходимые зависимости, выполнив команду: `pip install requests`
3. Создайте или поместите ваш файл отпечатка браузера (`fingerprint.json`) в ту же папку, где находится скрипт.
4. Запустите скрипт командой: `python wb.py`

---

## Формат файла fingerprint.json

Скрипт ожидает, что файл `fingerprint.json` содержит валидный JSON с параметрами браузера, которые требует текущая версия антибот-защиты.
